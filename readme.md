# User API Automation Tests

Automated testing for CRUD operations using Java, Cucumber, and RestAssured.

## ✅ Scenarios covered
- Create user (valid/invalid)
- Get user
- Update user
- Delete user
- Non-existing user handling

## 🛠 Setup
- Java 11+
- Maven
- API running at `http://localhost:8080/api`

## ▶️ Run tests
```bash
mvn test
```
